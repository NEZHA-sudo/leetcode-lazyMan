# 题目描述

2235. 两整数相加  
https://leetcode.cn/problems/add-two-integers/description/  

难度：简单
标签：无

给你两个整数 num1 和 num2，返回这两个整数的和。

示例 1：
输入：num1 = 12, num2 = 5
输出：17
解释：num1 是 12，num2 是 5 ，它们的和是 12 + 5 = 17 ，因此返回 17 。

示例 2：
输入：num1 = -10, num2 = 4
输出：-6
解释：num1 + num2 = -6 ，因此返回 -6 。

提示：
-100 <= num1, num2 <= 100

# python2

// 执行用时20ms，击败%
// 消耗内存12.4MB，击败%
```
class Solution(object):
    def sum(self, num1, num2):
        """
        :type num1: int
        :type num2: int
        :rtype: int
        """
        return num1+num2
```
