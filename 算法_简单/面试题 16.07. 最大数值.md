# 题目描述

面试题 16.07. 最大数值  
https://leetcode.cn/problems/maximum-lcci/description/  

难度：简单
标签：max()

编写一个方法，找出两个数字a和b中最大的那一个。不得使用if-else或其他比较运算符。

示例：
输入： a = 1, b = 2
输出： 2

# python2

// 执行用时0ms，击败100%
// 消耗内存12.32MB，击败31.25%
```
class Solution(object):
    def maximum(self, a, b):
        """
        :type a: int
        :type b: int
        :rtype: int
        """
        return max(a,b)
```
